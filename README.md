
## ✨ Características

- **Diseño responsivo** — adaptable a pantallas de escritorio, tablet y móvil mediante media queries
- **Variables CSS** — paleta de colores centralizada con `--color-primario`, `--color-terciario`, etc.
- **Tipografía Google Fonts** — fuentes *Krona One* (títulos) y *Montserrat* (cuerpo)
- **Navegación multi-página** — 4 páginas HTML enlazadas con navegación consistente
- **BEM en nomenclatura CSS** — clases organizadas con metodología Block-Element-Modifier
- **Sin dependencias JS** — HTML y CSS puro, carga instantánea

## 🚀 Cómo ejecutar el proyecto

No requiere instalación ni dependencias. Solo clona el repositorio y abre el archivo principal:

```bash
git clone https://github.com/imannolM/tars-landing.git
cd tars-landing
```

Luego abre `index.html` en tu navegador, o usa Live Server si trabajas con VS Code:

```
Click derecho en index.html → "Open with Live Server"
```
